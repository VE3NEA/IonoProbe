Settings Dialog

  • Run this program when Windows starts
  • Start in the System Tray
  • Auto Download Data
A combination of these three options switches IonoProbe to the fully automatic mode. The program will start automatically when you turn your computer on, hide in the System Tray and ensure that you always have the latest space weather indices and a continuous set of historical data.

When the Auto Download Data option is turned on, you can enable or disable automatic downloads of NOAA reports using the Including reports checkbox.


  • Always stay on top
When this check box is ticked, the IonoProbe window stays on top of all other windows, even if you switch to another program.


  • Play this sound when a storm starts
This option allows you to enable audio alerts and specify what sound will be played. Alerts are discussed in the Using Alerts section.

Two buttons make the selection of a sound easy:

- browse your hard drive for a WAV sound file

- try the selected sound.

IonoProbe comes with a default sound file, Alarm.wav, but you can use any other file on your disk.
