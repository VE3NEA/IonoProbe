Downloading Data


Download Engine

Ionoprobe uses Microsoft Internet Explorer as a download engine. It does not open the browser window, but it loads some IE modules in the background. Because of this, you do not have to tell Ionoprobe your proxy address, dial-up settings, authentication passwords, etc. Just make sure that you have the latest Internet Explorer and that it is properly configured. If you can browse the Internet with IE, you can download data with IonoProbe.



Download Modes

Iono probe can work in either the Auto-Download or Manual Download mode. If you have a permanent connection to the Internet or an unlimited dial-up connection, use Auto-Download at all times, this will ensure that you always have the latest space weather indices and a continuous set of historical data.

When Auto-Download is enabled, IonoProbe downloads data according to a carefully designed schedule that takes into account the update time and frequency of the files on the Web, and the amount of data in the files. The schedule itself is updated at regular intervals from the Ionoprobe Web site.

If Auto-Download is disabled, you still can initiate downloads manually using the commands described in the next section.

You can enable or disable the Auto-Download mode:

  • by ticking the Auto-Download Data checkbox in the Settings Dialog;

  • by clicking on the Download -> Auto-Download command in the Main Menu;

  • by clicking on Auto-Download in the System Tray icon's Popup Menu;

  • by double-clicking on the Download Status icon (see below).

When Auto-Download Data is enabled, you can specify if text reports will be included in the downloads or not, by ticking the Including reports checkbox in the Settings Dialog.



Download Commands

Download Latest Data
Download the current values of all parameters.

Download Recent Data
Since some files include data for a calendar day, downloading Latest Data shortly after 00:00 UTC will give you only a few values. Use the Download Recent Data command to retrieve data for the previous day. This command is executed automatically on program start if Auto-Download Data is enabled.

Download Old Data
IonoProbe keeps the data downloaded in the previous sessions so that you have a historical view of the parameters. However, if you are running the program for the first time or after a long break, then no historical data are available. In such cases the Download Old Data command comes handy.

Download Reports
If you disabled automatic downlaods of NOAA reports, you can use this menu command to download the latest reports manually.

Abort Download
Immediately stop all running downloads.

All commands listed above work in both download modes. E.g., you can issue the Download Recent Data command when you are in the Auto-download mode if you think that some recent data are missing.



Download Status

A small icon in the bottom-right corner of the window indicates the status of the download engine:

The following images are displayed:


- Auto-Download disabled;

- waiting for the next download cycle;

- downloading data;

- all downloads succeeded;

- some downloads failed;

- all downloads failed. Double-check your IE configuration.


The icon has a mouse-over hint that displays additional status information, including the number of downloaded files, the error count, and the time left to the next download.
