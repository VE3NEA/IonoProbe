Main Window


Opening the main window

IonoProbe is designed to work in the background while you are doing other things on your computer. It does not occupy any space on your Desktop, all you can see is a small icon in the System Tray:



Just click on that icon to open the IonoProbe window:




To hide Ionoprobe back to the System Tray, click on the button or use the File -> Hide Window command in the main menu.

You can exit from Ionoprobe at any time using the File -> Exit command in the main menu or the Exit command in the Tray icon's popup menu.



Adjusting the main window

The IonoProbe's window is very flexible, you can adjust it in many ways.


Resize Vertically. Drag the upper or lower border of the window to hide some of the graphs, or to display all graphs:




Re-order graphs. Drag the graphs vertically with your mouse to change their order. In combination with vertical resize, this allows you to display only the parameters that you are interested in:




Resize Horizontally. Drag the left or right border of the window to change the amount of data displayed:




You can further customize the behavior of the IonoProbe's main window by changing the options in the Settings Dialog.
